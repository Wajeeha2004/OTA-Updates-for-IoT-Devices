# 🔄 **OTA Updates for IoT (ESP8266/ESP32)**


## 🚀**Features:**

### 🔧 **ArduinoOTA** – Upload code via Arduino IDE over WiFi

### 🌐 **ElegantOTA** – Update firmware through a web browser (/update)

## 🧰 **Requirements:**

1. ESP8266 or ESP32

2. Arduino IDE

3. Libraries: ArduinoOTA, ESPAsyncWebServer, ElegantOTA
